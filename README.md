Данный проект реализует систему для работы с рецептами и списками покупок. Включает следующие классы:
- Ingredient - класс для представления ингредиента (название, количество, единица измерения)
- Recipe - класс для представления рецепта (название, список ингредиентов, масштабирование)
- DietaryRecipe - наследник Recipe, добавляющий диетическую категорию
- ShoppingList - класс для формирования списка покупок из одного или нескольких рецептов

Запустить тесты в терминале можно так:
python -m pytest test_recipes.py –v
